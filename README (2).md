# Solana Micro Donations

A simple crowdfunding dApp built on the Solana Devnet.

## Features

- Connect Phantom Wallet
- Create donation campaigns
- Send micro-donations in SOL
- Uses Solana Web3.js

## Technologies

- HTML
- JavaScript
- Solana Web3.js
- Phantom Wallet

## How to Run

1. Install Phantom Wallet
2. Switch to Devnet
3. Run the project

Open:

http://localhost:3000

## Example

Create a campaign and donate SOL using Phantom wallet.

## Network

Solana Devnet
